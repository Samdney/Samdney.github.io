For the beginning some formality. Given be the discrete exponential function (1)
g^dP \equiv x(mod(p)) = (dz)p + x
and the discrete logarithm (2)
dP = \log_{g}((dz)p + x)).
Key: g base , positive integer; p prime number, g < p; dP unknown exponent, null or positive integer; dz unknown integer; x positive integer, 0 <= x <= p-1
Furthermore we will assume, p can be written as
p := a_0 + \sum_i=1 a_i y_i,
with a_0, a_i integers and y_i:=y_i|_(c_i) integers under the constraint c_i.
